
# Generative AI Resume Screening and Skill Matching System

## Description
This project uses Generative AI to analyze resumes and compare them with job descriptions.
It helps recruiters quickly shortlist candidates.

## Features
- Upload Resume (PDF)
- Enter Job Description
- AI extracts skills from resume
- Match percentage calculation
- Shortlist or Reject recommendation

## Technologies Used
- Python
- Streamlit
- OpenAI API
- PyPDF2

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Add your OpenAI API key inside app.py

3. Run the project
streamlit run app.py
